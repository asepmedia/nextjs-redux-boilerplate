This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run development mode:

```bash
npm run dev
# or
yarn run dev
# or
pnpm run dev
```

## Includes
* React Redux
* React Icons
* Atomic Design
* Auto Routing
